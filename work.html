Voice controlled Light system

this project is built using Arduino Uno, Isield and c++. user can control the lights using voice or by sending SMS. access to the home lights is only given to the owner's phone number to prevent others from misusing it. to get the access to the home lights system to a phone number they need to enter the password and the owner is notified when a new user is added. every time home light controlled using SMS or voice owner is notified by SMS for security purposes
